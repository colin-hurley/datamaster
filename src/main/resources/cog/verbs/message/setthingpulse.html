SetThingPulse

Sets a pulse for a thing. A thing pulse acts like a regular pulse but will end when the thing dies. Only a cog that receives the thing's messages can set or receive a thing pulse. The pulse message is sent by the thing as an event message and will be sent to all of the thing's associated cogs. Syntax:

SetThingPulse(thing, flex_delay);

To stop the thing pulse, use:

SetThingPulse(same_thing, 0);
