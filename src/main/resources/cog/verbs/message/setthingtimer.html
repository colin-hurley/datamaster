SetThingTimer

Sets a timer event for a thing. This is similar to a timer set by SetTimer(), but it is treated as an event and the timer message will be sent to all of the thing's associated cogs. Only a cog that receives the thing's messages can set a thing timer for the thing. The timer will not be stopped if the thing dies. Syntax:

SetThingTimer(thing, flex_delay);

To stop the thing timer, use:

SetThingTimer(same_thing, 0);
