jkSetPersuasionInfo

Sets the persuasion settings for a creature. The persuasion effects will be created when the creature is given the 0x20 Jedi Knight Flag. Syntax:

jkSetPersuasionInfo(creature, min_stars, max_stars);

min_stars: This is the minimum number of stars that will be created in a regular burst.

max_stars: This is the maximum number to be created.

Notes: When persuasion is on, JK creates a burst of stars every second. But every dozen or so bursts, JK creates a burst with a large number of stars.
