jkSetTargetColors

Sets the colors of each ring in the targeting reticle. Each color value is the number of a color cell in the first sector's colormap.

To find the cell number of the color you want, open the appropriate cmp file with Chomper1. Hold the cursor over the cell of the color you want and Chomper will display the number in its status bar.

Because different levels use different cmps, you need to make sure that you use a color that is the same in all colormaps. Syntax:

jkSetTargetColors(color1, color2, color3);

1 Chomper is a .cmp editing program made by the Code Alliance.
