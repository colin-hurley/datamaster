SetThingPos

Sets the position of a thing. Syntax:

SetThingPos(thing, pos);

pos: The position is given in coordinate form.

The major problem with SetThingPos() is that it does not update the thing's sector when it moves it. That means if the thing is moved out of its current sector, there will be sector errors - sector thing counts will be wrong, you won't be able to see the object, if the camera is focused on this object you'll get HOM, etc.
