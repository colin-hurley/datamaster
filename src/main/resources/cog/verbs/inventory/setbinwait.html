SetBinWait

Makes a bin wait for the given delay until it can be activated. SetBinWait() does not affect the bin's cog. It only stops the bin from being activated. Syntax:

SetBinWait(player, bin, flex_delay);

When run, SetBinWait() will disrupt a force power bin's fire message and prevent it's hotkey from sending the activated message for a very short time. This seems to be an error with the verb.
