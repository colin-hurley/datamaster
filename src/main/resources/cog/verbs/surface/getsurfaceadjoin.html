GetSurfaceAdjoin

When two sectors adjoin, each sector must have a surface for the adjoin. These two surfaces form the adjoin as a whole. GetSurfaceAdjoin() will return the other surface in the adjoin. Syntax:

adjoin_surf2=GetSurfaceAdjoin(adjoin_surf1);

Make sure that you use GetSurfaceAdjoin() on an adjoin. The verb is stable, but if given a regular surface, it will crash the game.
