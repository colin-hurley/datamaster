FireProjectileData

Same as FireProjectile, but additionally sets the user data for the created projectile. Syntax:

projectile=FireProjectileData(fire_thing, template, sound, submode, offset_vec, error_vec, extra_flex, proj_flags, AA_FOVX, AA_FOVZ, user_data);

user_data: This is the user data that will be given to the created projectile
