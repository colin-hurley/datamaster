CreateThingAtPosOwner

Creates a thing at a position and sets its parent. If the template's model has an insert offset, then that offset will be added to the position. Syntax:

new_thing=CreateThingAtPosOwner(template, sector, position, 'p y r', parent);

sector: It is important to give the thing the correct sector to avoid sector problems. Use the Position's Sector code from the Tutorial Section if you cannot get the sector.

position: This is the position to create the thing at.

'p y r': This is the pitch, yaw, and roll for the created thing's orientation. You can use a vector variable or the 'p y r' form.

parent: This is the thing that will become the parent of the created thing

This verb was not used by LEC, but it does appear in a commented code block within the seeking rail detonator weapon cog.
