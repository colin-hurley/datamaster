NextThingInCone

Returns the next thing in view that is within a cone. FirstThingInCone() must run before NTIC() to set the FOV, distance, and mask settings. If FTIC() was not run first, NTIC() will return -1. NTIC() will also return -1 when there are no more valid things to return that meet the FOV, distance, and mask settings. Syntax:

thing=NextThingInCone();
