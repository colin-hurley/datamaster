ChangeAllSectorsLight

Adds a value to the light ('Extra Light') for all sectors in the level. Syntax:

ChangeAllSectorsLight(light);

light: The flex value to add to each sector's current light. Positive values increase the light, and negative numbers decrease it.

MotS will not let the light exceed 1.0 or fall below 0.0.

This verb was never used by LEC.
