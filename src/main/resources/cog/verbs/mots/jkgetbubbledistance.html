jkGetBubbleDistance

When a thing is inside a bubble, jkGetBubbleDistance() will return the distance from the thing to the center of the bubble. If the thing itself has a bubble, jkGetBubbleDistance() will return 0. When the thing is not inside a bubble, jkGetBubbleDistance() will return 1000000. Note that jkGetBubbleDistance() works with other thing types as well as players. Syntax:

distance=jkGetBubbleDistance(thing);
