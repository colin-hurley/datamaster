FireProjectileLocal

Same as FireProjectile, but the projectile will be created on the local system only, so its creation will not be synchronized to other machines in a Multiplayer game. Syntax:

projectile=FireProjectileLocal(fire_thing, template, sound, submode, offset_vec, error_vec, extra_flex, proj_flags, AA_FOVX, AA_FOVZ);

This verb was not used by LEC, but it does appear in a commented code block within the seeking rail detonator weapon cog.
