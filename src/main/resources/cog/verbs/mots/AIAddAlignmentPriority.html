AIAddAlignmentPriority

Sets AI alignment for the player. Syntax:

effect_handle=AIAddAlignmentPriority(alignment, priority);

When an alignment is added with AIAddAlignmentPriority, the alignment change will stay in effect until it is stopped with AIRemoveAlignmentPriority.

When multiple alignments with different priorities are set (in any order), the alignments with lower priority will have no effect until the highest priority alignment is stopped.

When multiple alignments with the same priority are set, the first one that was set will have priority over the others of the same priority until it is stopped.
