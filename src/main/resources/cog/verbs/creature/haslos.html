HasLOS

Tests to see if a target is in view of a look_thing and returns a boolean value. The look_thing does not have to be facing the target for HasLOS() to return true. Syntax:

boolean=HasLOS(look_thing, target);

HasLOS will return false if view of the target thing is obstructed by another thing of type 'cog' (e.g. doors, elevators, and other 3DO-based geometry). It will also return false if the target thing is behind an adjoined surface that does not have the 0x1 adjoin flag set. If HasLOS is given a reference to a non-existent thing, it will return -1.
