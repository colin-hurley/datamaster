ActivateBin

Despite its name, ActivateBin() does not literally activate a bin. Instead, it runs the fire message in force power cogs. After the fire message runs, JK waits the length of the firewait and then calls the fire message again. The fire message will continue to loop until DeactivateBin() is used. Syntax:

ActivateBin(player, firewait, force_bin);

firewait: This is the time to wait before the fire message can run again.

force_bin: This is the bin number of the force power. This bin number can be obtained from the senderref of the activated message.
