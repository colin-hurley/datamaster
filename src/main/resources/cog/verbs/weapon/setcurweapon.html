SetCurWeapon

Sets the current weapon. Syntax:

SetCurWeapon(player, weap_num);

Making the weapon "current" tells the engine to which weapon the weapon-related messages should be sent, including activated, fire, deactivated, deselected, and autoselect. In addition, the HUD ammo counter will display the value of the ammo bin that is associated with the current weapon's bin number (this association is hard-coded in the game engine; for more details, refer to the list of special bins in the Inventory Verbs' section notes).

Note that SetCurWeapon() wants a weapon number and not a bin number. For example, if you add a new weapon with bin 116 (0x4 bin flag), you can use SetCurWeapon(player, 11) to select it.
