Preblock

Preblock is sent by an actor or player when it is trying to deflect a projectile with the lightsaber. Preblock must return a value with ReturnEx() to indicate whether the projectile should be deflected or not. If 0 is returned, the projectile will not be deflected. If any other value is returned, the projectile will be deflected. If none of the thing's associated cogs handle the preblock message, the projectile will not be deflected.

SenderThe creature that is deflecting the projectile.
