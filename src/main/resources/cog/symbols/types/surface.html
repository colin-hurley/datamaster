Surface Symbol Type

The surface symbol type is used to store the reference number of a surface. If the surface's value is passed from a JKL, any messages sent by the surface will be received by the symbol's cog. Syntax: 
surface     surf1

Mask Info: Surfaces can send several messages to their cogs. Giving the surface Mask Flags allows the cog to ignore messages from the surface caused by things that were excluded by the mask. Surfaces also use default masks, perhaps 0x400.
