0x2 Jedi Knight Flag

Even while jkEnableSaber() has enabled the saber, it cannot collide until the 0x2 Jedi Knight Flag has been set. Markers in the keyframe trigger this flag to be set and cleared. Marker 4 sets the flag, and Marker 5 clears it. Creatures using the saber can only damage a thing once for each time the 0x2 flag is set.
