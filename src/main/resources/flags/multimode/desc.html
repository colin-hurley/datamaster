MP Mode Flags

Multiplayer Mode Flags control the functions of a game related to multiplayer. These flags are heavily relied on by the CTF system and may have been created expressly for CTF gameplay.

Some of these mode flags have their limitations - such as having to be set on startup or not being changeable at all. But if not mentioned otherwise, JK will look at these flags when the related event occurs. So you can set and clear the flag ingame.
