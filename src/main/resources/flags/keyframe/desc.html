Keyframe Flags

Keyframe Flags are used to control things' keyframe animations. They are set when the animation is started and cannot be modified or retrieved afterwards. All of these flags with the exception of 0x10 are used for both internal and external animations.

With PlayKey() and jkPlayPOVKey(), you can reference the key file and set the flags with the verb. But with other verbs such as PlayMode(), the number of the submode in the thing's puppet file is used. And the Keyframe Flags set in the puppet are used.
