Inventory Flags
HexDecPurpose
0x11Unknown.
0x22Bin is used by an item.
0x44Bin is used by a weapon.
0x88Bin is used by a force power.
0x1016Bin will call the damaged message for the player in its cog.
0x2032Bin is available by default.
0x4064Bin's value is reset for the next level in an episode.
0x80128Bin will be added to a backpack.
0x100512Bin can be assigned a hotkey.
